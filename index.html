Neural Style Transfer

Introduction

We will build a neural style transfer model, a model to take a content image and a style image and output the content image with the style from the style image applied to it.

Algorithm

We will use the VGGNet model that is pre-trained on ImageNet. We will initialize the combined image to the content image, and we will pass the content image, style image, and combined image to the model. The model will tune the pixel values of the combined image until it represents a combination of the content image and the style image. Since we will have a content loss and a style loss, we will stop training the model when the sum of both losses is minimized.
